# dx翻译

别名dx翻译,m-deeplx,支持各种自己部署服务的utools翻译插件,适合有点运维基础的人员使用

## 功能特点

- 私有deeplx部署翻译
- 私有chatgpt部署翻译(待支持)
- 私有谷歌部署翻译(待支持)
- 私有Amazon部署翻译(待支持)
- 双语菜单支持
